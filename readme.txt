Per eseguire l'applicazione .jar  da propt dei comandi è necessario:

-  digitare il comando java -jar seguito da uno spazio, per poi inserire la directory del file .jar
 	esempio: "C:\Desktop\Adrenaline.jar"
 successivamente premere invio


Nell'applicazione sono state implementate le seguenti funzioni:

- GUI interface
- Socket comunication
- Regole complete del gioco
